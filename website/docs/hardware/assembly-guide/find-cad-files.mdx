# Step 0: Find CAD Files

We are open-sourcing the CAD files for this project to encourage collaboration, transparency, and ease of customization.

You can access all 3D models, assemblies, and documents directly from our GitHub repository:

👉 **[Open-source CAD on GitHub](https://github.com/enactic/openarm_hardware)**

## 📁 What’s Included

The repository includes:

- ✅ STEP Files: For manufacturing and modification
- ✅ STL Files: Ready-to-print 3D models for all 3D-printed components
- ✅ Sample Assembly to help you get started

## 🔄 Updating & Contributions

We regularly update the CAD files as improvements are made to the hardware.
If you find issues or want to contribute:

- 🛠 Submit a pull request
- 💬 Open a GitHub issue 
- 📧 Contact us directly on **[Discord](https://discord.gg/tpnKxHuJY3)**
