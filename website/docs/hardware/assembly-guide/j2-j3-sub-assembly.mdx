# Step 3: J2-J3 Sub-Assembly

:::warning
### Prerequisites
**Before starting** the assembly, please configure motor IDs by following [Step 1: Setup Motor ID](/software/setup/motor-id).
:::

1. Attach `J2_A` to `J2_B` using 2 `M4x10` bolts
<AssemblyGuideImage src="j2-j3/step1.png" alt="step1" width="40%" />

2. Carefully place `J2_C` over `J2_A` and `J2_B` such that the holes align. Attach `J2_C` to `J2_A` using 4 `M4x6` bolts
<AssemblyGuideImage src="j2-j3/step2.png" alt="step2" width="40%" />

3. Attach `J2_C` to `J2_B` using 4 `M4x6` bolts
<AssemblyGuideImage src="j2-j3/step3.png" alt="step3" width="40%" />

:::info
That concludes the J2-J3 Sub-Assembly
<AssemblyGuideImage src="j2-j3/conclusion.png" alt="conclusion" width="40%" />
:::
