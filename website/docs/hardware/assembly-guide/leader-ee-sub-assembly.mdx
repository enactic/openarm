# Step 7.2: Leader End-Effector Sub-Assembly

1. Attach `J8_A` to `J8_B` using 3 `M3x6` bolts

<AssemblyGuideImage src="leader-ee/step1.png" alt="step1" width="30%" />

2. Insert two `M4x4 heat set inserts` in the `swivel_rotor_leader` using a soldering iron.

<AssemblyGuideImage src="leader-ee/swivel-rotor-heat-set-insert.png" alt="swivel-rotor-heat-set" width="50%" />

3. Insert one `M4x4 heat set insert` in the `right_pincer` and `left_pincer` each using a soldering iron.

<AssemblyGuideImage src="leader-ee/pincer-heat-set-insert.png" alt="pincer-heat-set" width="30%" />

4. Insert 4 `M3x6 heat set inserts` in the `rail_connector_leader` using a soldering iron.

<AssemblyGuideImage src="leader-ee/rail-connector-heat-set-insert.png" alt="rail-connector-heat-set" width="50%" />

5. Carefully insert bearings `MR126ZZ` into `swivel_link_leader`. Note that two bearings need to be inserted from opposite directions.

<AssemblyGuideImage src="leader-ee/bearings.png" alt="bearings" width="30%" />

6. Attach `rail_connector_leader` to the `J8` motor using 4 `M3x8` bolts as shown in the figure. Ensure that the flanges of `rail_connector_leader` are aligned to the two Power+CAN ports of `J8`

<AssemblyGuideImage src="leader-ee/step2.png" alt="step2" width="50%" />

7. Insert two `M3x15` bolts along the flanges to fasten `rail_connector_leader` to `J8` motor.

<AssemblyGuideImage src="leader-ee/step3.png" alt="step3" width="50%" />

8. Connect `swivel_rotor_leader` to the rotor of `J8` using 4 `M3x12` bolts.

<AssemblyGuideImage src="leader-ee/step4.png" alt="step4" width="60%" />

9. Attach the two `swivel_link_leader` components to the `swivel_rotor_leader` using step bolts `DBSY4-5-4`. Make sure the bearings are oriented such that the fully exposed (open) side of each bearing faces toward the `swivel_rotor_leader`, as shown in the image. This orientation ensures proper alignment and freedom of movement for the linkages.

<AssemblyGuideImage src="leader-ee/step5.png" alt="step5" width="60%" />

7. Attach the linear guide to `rail_connector_leader` using 4 `M3x6` bolts.

<AssemblyGuideImage src="leader-ee/step6.png" alt="step6" width="60%" />

8. Attach the `left_pincer` and `right_pincer` on the slider blocks using 2 `M3x6` bolts each. Ensure that the heat set insert is towards the `swivel_link_leader` for both the pincers. 

<AssemblyGuideImage src="leader-ee/step7.png" alt="step7" width="60%" />

9. Attach the `swivel_link_leader` components to the `left_pincer` and the `right_pincer` using step bolts `DBSY4-5-4`.

<AssemblyGuideImage src="leader-ee/step8.png" alt="step8" width="60%" />

10. Mount the subassembly created in Step 1 to the rear face of the `J8` motor using four `M3Ã—10` bolts.

- When assembling the right arm, ensure the `J8_A` part is aligned with the `right pincer`.
- When assembling the left arm, align `J8_A` with the `left pincer`.

The alignment ensures proper mechanical coupling and correct orientation of the jaws during operation, as indicated by the arrows in the image.

<AssemblyGuideImage src="leader-ee/step9.png" alt="step10" width="70%" />
